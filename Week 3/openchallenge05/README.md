float를 이용해서 각 요소들을 배치함

nav와 section은 블록요소 이지만 display: inline-block;으로 nav와 section을 나란히 배치할 수 있게함.

```css
nav {
    background-color: orange;
    float: left;
    height: 72%;
    display: inline-block;
    width: 15%;
}
section {
    background-color: lightblue;
    float: left;
    width: 85%;
    left: 15%;
    height: 72%;
    display: inline-block;
}
```
